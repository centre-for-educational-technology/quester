# Quester
 
Quester is a tool for collecting feedback on your lesson.

## Requirements

Quester uses Laravel v9 and requires PHP version 8

## Installation

 #### Configure .env file

 * cp .env.example .env
 * php artisan key:generate
 
 #### Install dependencies
 
 * composer install
 
 #### Install Node dependencies
 
 * npm install
 * npm run prod
 
 #### Run migrations
 
 * php artisan migrate
